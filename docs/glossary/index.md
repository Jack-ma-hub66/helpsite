# 词汇表

帮助您快速理解教程中出现的专业术语。

---

## 订阅相关

### 订阅链接（Subscribe URL）

**定义：** 您在服务商用户中心获取的唯一 URL，客户端通过该链接自动同步最新的节点配置。

**注意：** 订阅链接是私密的，请勿泄露给他人，否则他人可消耗您的流量。

---

### 更新订阅

**定义：** 重新从订阅链接拉取最新的节点配置文件。

**什么时候需要更新：**

- 服务商通知需要更新订阅
- 节点大批量失效或超时
- 配置了新节点后想同步

**操作方法：** 在客户端软件内找到已导入的配置，选择「更新」或「刷新」。

---

### 重置 URL

**定义：** 使旧的订阅链接失效，并生成一条新的唯一链接。

**使用场景：** 订阅链接泄露后，立即重置可防止他人继续使用。

!!! warning "操作不可逆"
    重置后旧链接立即失效，所有已配置该链接的设备都需要重新导入新链接。

---

### UUID

**定义：** 节点身份验证的密钥，用于向代理服务器证明身份。

**使用场景：** 怀疑账号安全时可重置 UUID。重置后需重新更新订阅才能生效。

---

## 代理模式

### 规则模式（Rule）

根据预设规则智能分流：

- 国内网站 → 直接连接（不经代理，速度更快）
- 被屏蔽网站 → 走代理

**推荐大多数用户使用。**

---

### 全局模式（Global）

所有网络流量都通过代理服务器。国内网站速度可能变慢，但可确保所有访问都走代理。

---

### 直连模式（Direct）

禁用代理，所有流量直连。等同于没有开启代理软件。

---

## 网络相关

### 节点（Node）

代理服务器的具体连接点，通常对应一个物理服务器。不同节点有不同的延迟和带宽。

---

### 延迟（Latency）

数据从您的设备到代理服务器的往返时间，单位毫秒（ms）。

| 延迟 | 体验 |
|------|------|
| < 100ms | 优秀 |
| 100–200ms | 良好 |
| 200–500ms | 一般 |
| > 500ms | 较差，建议换节点 |

---

### 负载均衡（Load Balance）

自动将流量分散到多个节点，避免单一节点过载，提升整体速度和稳定性。

---

### 系统代理（System Proxy）

将代理设置注入操作系统，使系统中所有应用（浏览器、下载工具等）都走代理，无需每个应用单独设置。

---

## 文件格式

### .yaml / .yml

节点配置文件的格式，Clash 系列客户端（Clash Verge、OpenClash）使用此格式。

### .json

Sing Box 使用的配置文件格式。

---

## 操作相关

### Assets（GitHub）

在 GitHub Releases 页面，点击「Assets」可展开文件下载列表，从中选择适合您系统的安装包。

### WebView2

Microsoft 提供的 Web 渲染组件，Clash Verge 在 Windows 上运行时需要此组件。如果缺少，从 [Microsoft 官网](https://developer.microsoft.com/en-us/microsoft-edge/webview2/) 下载安装。
