# Windows 平台

适用于 Windows 10 / 11 的代理客户端配置教程。

## 推荐客户端

| 软件名称 | 特点 | 教程 | 下载 |
|---------|------|------|------|
| **Clash Verge Rev** | 图形界面，操作简单，适合新手 | [查看教程](clash-verge/quick-start.md) | [GitHub](https://github.com/clash-verge-rev/clash-verge-rev/releases) |
| **v2rayN** | 功能强大，支持多核心，适合进阶用户 | [查看教程](v2rayn/quick-start.md) | [GitHub](https://github.com/2dust/v2rayN/releases) |

!!! note "新手推荐"
    如果您是第一次使用，推荐从 **Clash Verge Rev** 开始，界面友好，配置步骤简单。

## 系统要求

- Windows 10（版本 1903 及以上）或 Windows 11
- x86 / x64 架构处理器
- 已安装 [Microsoft Edge WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)（Clash Verge 需要）
