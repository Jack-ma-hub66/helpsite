# v2rayN — 快速开始

v2rayN 是一款支持 Xray 核心和 sing-box 核心的 Windows 代理客户端，功能强大，适合进阶用户使用。

---

## 第一步：下载与安装

前往 [v2rayN GitHub Releases 页面](https://github.com/2dust/v2rayN/releases)，下载以下版本：

```
v2rayN-windows-64-With-Core.zip    ← 包含所有核心文件，推荐首次使用
```

!!! warning "注意解压路径"
    请将文件解压到**非系统盘**的目录，例如 `D:\v2rayN\`，避免权限问题。

安装步骤：

1. 解压下载的压缩包
2. 双击 `v2rayN.exe` 启动程序
3. 如弹出防火墙提示，选择「**允许访问**」

---

## 第二步：设置语言

1. 点击菜单栏「**设置**」→「**参数设置**」
2. 找到「**语言**」，选择「**简体中文**」
3. 点击「确定」后重启软件

---

## 第三步：添加订阅节点

1. 从服务提供商处复制**订阅链接**
2. 在 v2rayN 中点击菜单「**订阅分组**」→「**订阅分组设置**」
3. 点击「**添加**」，填写以下信息：
    - **别名**：填入便于识别的名称（例如 `MyService`）
    - **地址（URL）**：粘贴订阅链接
4. 点击「确定」保存
5. 返回主界面，点击「**订阅分组**」→「**更新全部订阅（不通过代理）**」，等待更新完成

---

## 第四步：启用代理

1. 在节点列表中右键点击所需服务器，选择「**设为活动服务器**」（被激活的节点显示**加粗**）
2. 在任务栏托盘区右键点击 v2rayN 图标
3. 选择「**自动配置系统代理**」

根据需求选择路由模式：

| 模式 | 说明 |
|------|------|
| 绕过大陆 | 仅代理被屏蔽网站，国内流量直连（推荐） |
| 全局模式 | 所有流量通过代理 |

---

## 第五步：测试连接

打开浏览器访问测试网站，查看主界面底部日志确认连接状态正常。

**常见问题快速排查：**

- 无法连接 → 检查节点是否被重置过 UUID，重新导入订阅
- 速度缓慢 → 切换其他节点，检查本地网络带宽
- 提示「核心文件未找到」→ 删除旧目录，重新下载 `With-Core` 版本
