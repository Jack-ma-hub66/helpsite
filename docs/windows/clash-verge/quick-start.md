# Clash Verge Rev — 快速开始

Clash Verge Rev 是一款功能强大的图形化代理客户端，支持多种协议，适合 Windows 新手用户使用。

---

## 第一步：下载与安装

### 下载软件

前往 [Clash Verge Rev GitHub Releases 页面](https://github.com/clash-verge-rev/clash-verge-rev/releases)，下载适用于 Windows 的安装包：

```
Clash.Verge_x.x.x_x64-setup.exe    ← 64 位系统（推荐）
Clash.Verge_x.x.x_x86-setup.exe    ← 32 位系统
```

!!! tip "如何找到下载链接"
    在 GitHub Releases 页面，点击 **Assets** 展开文件列表，再点击对应文件下载。

### 安装软件

1. 双击下载的 `.exe` 文件
2. 按提示选择安装路径（**建议使用默认路径，且路径不要包含中文字符**）
3. 点击「下一步」等待安装完成
4. 安装后可通过开始菜单或桌面快捷方式启动

---

## 第二步：导入订阅配置

Clash Verge 支持两种导入方式：

=== "方法一：订阅链接（推荐）"

    1. 从您的服务提供商处复制**订阅链接**
    2. 打开 Clash Verge，点击左侧菜单「**配置**」
    3. 点击「**订阅链接**」→「**添加订阅**」
    4. 粘贴订阅链接，点击「**保存**」
    5. 软件自动更新并显示节点列表

=== "方法二：本地文件（备用）"

    如果订阅链接导入失败，可以手动下载配置文件：

    1. 在订阅链接末尾添加 `&app=clashmeta`，用浏览器打开
    2. 浏览器会自动下载配置文件
    3. 将文件后缀重命名为 `.yaml`
    4. 在 Clash Verge 中点击「**导入文件**」，选择该文件

---

## 第三步：启用代理

1. 点击左侧菜单「**设置**」
2. 找到「**系统代理**」，将其开启
3. 在「**节点选择**」页面选择代理模式：

| 模式 | 说明 |
|------|------|
| 规则模式 | 根据规则智能分流（**推荐新手使用**） |
| 全局模式 | 所有流量通过代理 |
| 直连模式 | 禁用代理 |

---

## 第四步：选择节点并测试

1. 打开「**节点选择**」，在服务器列表中选择一个节点
2. 打开浏览器访问测试网站，确认是否正常
3. 正常连接的节点显示**绿色状态**，延迟应低于 200ms
