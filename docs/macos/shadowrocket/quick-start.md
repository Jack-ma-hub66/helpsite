# Shadowrocket（小火箭）— 快速开始（macOS）

Shadowrocket 是功能强大的付费代理客户端，支持多种协议，界面简洁。

---

## 第一步：购买与安装

Shadowrocket 售价 **$2.99 美元**，需从 App Store 购买。

!!! warning "账号要求"
    需要**非中国大陆区**的 Apple ID（美区、港区、台湾区均可）。购买后同一账号下的所有 Apple 设备均可免费安装。

---

## 第二步：添加订阅

1. 打开 Shadowrocket，点击右上角「**+**」
2. 选择类型为「**Subscribe**」
3. 粘贴订阅链接，点击「完成」
4. 点击订阅名称旁的刷新图标更新节点

---

## 第三步：启用代理

1. 点击顶部「**未连接**」开关，开启代理
2. 系统弹出 VPN 权限请求，点击「允许」并验证密码
3. 在节点列表中选择所需节点
