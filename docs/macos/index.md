# macOS 平台

适用于 Intel 和 Apple Silicon (M 系列) Mac 设备的客户端配置教程。

## 推荐客户端

| 软件名称 | 特点 | 来源 | 教程 |
|---------|------|------|------|
| **Clash Verge Rev** | 图形界面，操作简单 | GitHub 下载 | [查看教程](clash-verge/quick-start.md) |
| **Sing Box** | 支持新一代协议，体积小 | App Store | [查看教程](singbox/quick-start.md) |
| **Shadowrocket（小火箭）** | 功能丰富，付费应用 | App Store（需美区账号） | [查看教程](shadowrocket/quick-start.md) |

!!! note "Apple Silicon 兼容性"
    以上三款客户端均原生支持 Apple Silicon（M1 / M2 / M3 / M4）芯片，无需 Rosetta 转译。
