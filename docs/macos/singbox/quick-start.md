# Sing Box — 快速开始（macOS）

Sing Box 是一款支持新一代协议的轻量代理客户端，可从 App Store 直接安装。

---

## 第一步：下载安装

在 App Store 搜索「**Sing Box**」或访问：
[https://apps.apple.com/us/app/sing-box-vt/id6673731168](https://apps.apple.com/us/app/sing-box-vt/id6673731168)

!!! warning "区域限制"
    中国大陆 App Store 账号无法下载，需使用**美区或港区**账号。

---

## 第二步：导入订阅

1. 从服务提供商处复制**订阅链接**
2. 打开 Sing Box，点击右上角「**+**」
3. 选择「**从 URL 导入**」，粘贴订阅链接
4. 点击「**保存**」，等待节点加载完成

---

## 第三步：启用代理

1. 点击「**启动**」按钮
2. 系统弹出 VPN 权限请求，点击「**允许**」
3. 选择合适的节点，确认连接成功
