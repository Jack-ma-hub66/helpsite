# 常见问题

涵盖所有平台的通用问题。客户端专属问题请查看对应教程页面内的「常见问题」章节。

---

## 连接问题

??? question "订阅链接导入失败"
    **可能原因：** 本地网络屏蔽了订阅服务器。

    **解决方案：**

    1. 切换到手机热点或其他网络后重试
    2. 联系服务提供商获取新的订阅链接
    3. 使用本地文件导入方式（参见各平台教程的备用导入方法）

??? question "节点全部超时或无法连接"
    **解决方案：**

    1. 点击「更新订阅」获取最新节点列表
    2. 切换到延迟较低的节点
    3. 如全部节点均失败，联系服务提供商排查线路问题

??? question "关闭代理软件后，无法上网"
    代理软件异常退出后，系统代理设置可能残留。

    **Windows 解决方法：**
    按 `Win + R` 输入 `inetcpl.cpl` → 连接 → 局域网设置 → 取消勾选「为 LAN 使用代理服务器」

    **macOS 解决方法：**
    系统设置 → 网络 → 当前网络 → 详细信息 → 代理 → 取消所有代理勾选

## 速度问题

??? question "连接后网速很慢"
    **排查步骤：**

    1. 切换到其他节点（选延迟低的）
    2. 将代理模式改为「规则」而非「全局」，减少不必要的代理流量
    3. 检查本地网络是否有其他设备占用大量带宽
    4. 尝试不同时段连接（高峰时段节点拥挤是正常现象）

??? question "看视频/游戏卡，但网页浏览正常"
    视频流媒体和游戏对延迟更敏感。建议：

    - 选择专用的流媒体或游戏节点（如服务商提供了分类节点）
    - 选择地理位置更近的节点
    - 开启 UDP 转发（Shadowrocket、Sing Box 均有此选项）

## 账号与安全

??? question "订阅链接泄露了怎么办"
    立即登录用户中心，执行以下操作：

    1. **重置 URL**：旧链接立即失效，生成新链接
    2. **重置 UUID**：更新节点密钥
    3. 在所有设备上重新导入新的订阅链接

    详见[词汇表 → 重置 URL](../glossary/index.md)。

??? question "多台设备可以同时使用吗"
    取决于您的套餐限制。通常订阅链接本身不限制设备数量，但服务商可能对同时在线设备数有限制，请查看您的套餐说明。

## App Store 相关（iOS / macOS）

??? question "如何注册美区 Apple ID"
    基本步骤：

    1. 退出当前 Apple ID（设置 → Apple ID → 退出登录）
    2. 在 App Store 点击右上角头像 → 退出
    3. 用美区账号登录（需提前注册好美区账号）

    !!! tip
        建议不要在「设置」中切换 Apple ID（会影响 iCloud），仅在 **App Store** 内切换账号即可。

??? question "购买了 App，换设备需要重新付费吗"
    不需要。同一 Apple ID 下的所有设备可免费重新下载已购 App。但不同 Apple ID 的购买记录无法共享。
