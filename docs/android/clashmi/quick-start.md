# ClashMi（Clash Meta）— 快速开始（Android）

ClashMi 是 Android 平台上操作界面最友好的代理客户端之一，基于 Clash Meta 核心。

---

## 第一步：下载安装

前往 [ClashMi GitHub Releases](https://github.com/clashmi/clashmi/releases)，下载 `.apk` 文件。

!!! tip "安装提示"
    下载后系统会提示「未知来源」安装风险，在「设置」→「安全」中允许安装即可。安装完成后建议关闭该选项。

---

## 第二步：导入订阅

1. 从服务提供商处复制**订阅链接**
2. 打开 ClashMi，点击底部「**配置**」
3. 点击右上角「**+**」→「**URL 导入**」
4. 粘贴订阅链接，填写名称，点击「**保存**」
5. 点击刚添加的配置使其生效（出现对勾标记）

---

## 第三步：启用代理

1. 回到主界面，点击「**启动**」
2. 系统弹出 VPN 连接请求，点击「**确定**」
3. 在「代理」页面选择节点和代理模式

| 模式 | 说明 |
|------|------|
| 规则 | 智能分流，国内直连，国外走代理（推荐） |
| 全局 | 所有流量走代理 |
| 直连 | 禁用代理 |

---

## 第四步：测试连接

打开浏览器访问测试网站，确认代理正常工作。
