# Sing Box — 快速开始（Android）

Sing Box 是跨平台的轻量级代理客户端，支持最新代理协议。

---

## 第一步：下载安装

=== "Google Play"
    在 Google Play 搜索「**Sing Box**」安装（需科学上网）

=== "GitHub 下载"
    前往 [Sing Box GitHub Releases](https://github.com/SagerNet/sing-box/releases) 下载 `.apk` 安装包

---

## 第二步：导入订阅

1. 复制服务提供商的**订阅链接**
2. 打开 Sing Box，点击「**+**」
3. 选择「**从剪贴板导入**」或手动粘贴链接
4. 保存后点击配置名称使其激活

---

## 第三步：启用代理

1. 点击主界面的「**启动**」按钮
2. 允许 VPN 连接请求
3. 确认状态栏出现 VPN 图标，表示代理已启用
