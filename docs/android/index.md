# Android 平台

适用于大多数基于 Android 系统的手机和平板设备。

## 推荐客户端

| 软件名称 | 特点 | 教程 |
|---------|------|------|
| **ClashMi（Clash Meta）** | 图形界面，操作简单，新手友好 | [查看教程](clashmi/quick-start.md) |
| **Sing Box** | 支持新一代协议，轻量 | [查看教程](singbox/quick-start.md) |

!!! tip "下载来源说明"
    Android 客户端可通过 GitHub Releases 或 Google Play 下载。无法访问 Play Store 时，直接从 GitHub 下载 `.apk` 文件安装。
