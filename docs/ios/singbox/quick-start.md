# Sing Box — 快速开始（iOS / iPadOS）

Sing Box 是免费的代理客户端，支持 iPhone 和 iPad，兼容最新代理协议。

---

## 第一步：下载安装

在 App Store 搜索「**Sing Box**」，或直接访问：
[https://apps.apple.com/us/app/sing-box-vt/id6673731168](https://apps.apple.com/us/app/sing-box-vt/id6673731168)

!!! warning "区域限制"
    中国大陆 App Store 无法搜索到该应用，需切换至**美区、港区或台湾区**账号下载。

---

## 第二步：导入订阅

1. 从服务提供商处复制**订阅链接**
2. 打开 Sing Box，点击右上角「**+**」
3. 选择「**从 URL 导入**」，粘贴链接
4. 点击「**保存**」，等待节点列表加载

---

## 第三步：启用代理

1. 点击主界面「**启动**」
2. 系统弹出「要添加 VPN 配置吗」，点击「**允许**」
3. 使用 Face ID / Touch ID 或密码验证
4. 状态栏出现 VPN 图标，代理启用成功

---

## 常见问题

**Q: 连接后网络反而变慢？**
切换到延迟更低的节点，或将代理模式改为「规则」（仅代理需要的流量）。

**Q: 图标消失 / 自动断开？**
iOS 系统会在省电模式下关闭 VPN，进入「设置」→「VPN」→ 开启「按需连接」可解决。
