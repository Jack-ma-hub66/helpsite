# Shadowrocket（小火箭）— 快速开始（iOS / iPadOS）

Shadowrocket 是 iOS 平台上功能最全面的代理客户端之一，一次购买，全设备通用。

---

## 第一步：购买与安装

- **价格**：$2.99 美元
- **下载地址**：App Store 搜索「**Shadowrocket**」
- **账号要求**：需要非中国大陆区 Apple ID（美区、港区、台湾区）

!!! tip "共享购买"
    同一 Apple ID 登录的所有 iOS / iPadOS 设备均可免费安装，无需重复购买。

---

## 第二步：添加订阅

1. 打开 Shadowrocket，点击右上角「**+**」
2. 类型选择「**Subscribe**」
3. 在 URL 栏粘贴**订阅链接**
4. 点击「**完成**」，等待节点加载
5. 长按订阅条目可手动刷新节点

---

## 第三步：启用代理

1. 打开 Shadowrocket 首页，点击顶部开关（显示「未连接」）
2. 系统请求添加 VPN 配置，点击「**允许**」
3. 验证密码或 Face ID
4. 开关变绿，代理启用成功

---

## 推荐配置

| 设置项 | 推荐值 |
|-------|-------|
| 代理模式 | 规则（Rule） |
| 测速方式 | ICMP（延迟测试更准确） |
| UDP 转发 | 开启 |
