# Passwall2 — 快速开始

Passwall2 是 OpenWrt 上轻量级的代理插件，配置相对简单，适合不需要复杂规则的用户。

---

## 第一步：安装 Passwall2

1. 在 OpenWrt 后台，进入「**系统**」→「**软件包**」
2. 添加 Passwall2 软件源或直接上传 `.ipk` 安装包
3. 安装完成后在「**服务**」菜单找到「**PassWall 2**」

---

## 第二步：添加订阅

1. 进入 Passwall2 管理界面
2. 点击「**节点订阅**」→「**新建订阅**」
3. 粘贴订阅链接，保存
4. 点击「**手动更新所有订阅**」，等待节点导入

---

## 第三步：基本配置

1. 进入「**基本设置**」
2. 开启「**主开关**」
3. 设置代理模式：
    - **绕过大陆 IP**：国内直连，国外走代理（推荐）
    - **全局代理**：所有流量走代理
4. 选择一个可用节点
5. 点击「**保存并应用**」

---

## 验证是否生效

用局域网内的设备访问测试网站，确认代理正常工作。
