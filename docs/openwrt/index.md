# OpenWrt 平台

适用于主流路由器和软路由设备，配置完成后可为局域网内所有设备提供代理服务，无需在每台设备单独配置。

## 推荐客户端

| 软件名称 | 特点 | 教程 |
|---------|------|------|
| **OpenClash** | 功能全面，可视化界面 | [查看教程](openclash/quick-start.md) |
| **Passwall2** | 轻量，配置简单 | [查看教程](passwall2/quick-start.md) |

!!! warning "适用人群"
    OpenWrt 配置需要一定网络基础知识。普通用户建议优先使用手机或电脑客户端。
